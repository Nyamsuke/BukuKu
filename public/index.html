TENTANG BUKUKU

Bukuku adalah aplikasi berbasis web yang dirancang untuk membantu pengguna menemukan informasi buku hanya melalui gambar sampul buku. Sistem memanfaatkan teknologi Optical Character Recognition (OCR) menggunakan Google Vision API untuk membaca teks pada cover buku, kemudian mencocokkannya dengan database Open Library API.

Aplikasi ini dikembangkan karena proses pencarian buku secara manual sering kali membutuhkan waktu lebih lama, terutama ketika pengguna hanya mengetahui tampilan cover buku tanpa mengetahui judul maupun penulisnya. Selain menampilkan informasi buku seperti judul, penulis, penerbit, dan tahun terbit, Bukuku juga menyediakan rekomendasi buku terkait berdasarkan kategori maupun penulis yang serupa.

Sistem dibangun menggunakan HTML, CSS, JavaScript, Node.js, serta integrasi API berbasis cloud agar proses identifikasi buku menjadi lebih cepat, praktis, dan efisien.

Cara Kerja Sistem

User Upload Gambar → Ekstraksi Teks (OCR) → Text Processing → Pencarian Open Library API → Informasi & Rekomendasi Buku
